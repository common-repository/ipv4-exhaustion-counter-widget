Wordpress IPv4 Exhaustion Counter is a widget that visualizes the status of IPv4 address exhaustion.

== Description ==

Wordpress IPv4 Exhaustion Counter is a widget that visualizes the status of IPv4 address exhaustion which mashed up with the "IPv4 Global Unicast Address Assignments" provided by IANA and "IPv4 Address Report" researched by Mr. Geoff Huston of APNIC.

== Credits ==

The IPv4 Exhaustion Counter itself has been created by Inetcore. The widget is only a small wrapper around the API call of the counter.

== Installation ==

Installing should be a piece of cake and take fewer than five minutes.

You can either install the plugin with three different methods. The procedures are listed by ease of installation.

1. Automated
2. Half-Automated
3. Manual

= The automated Installation =

1. Go to the plugin page in the admin pane. Search for "IPv4 Exhaustion" and install the plugin.
2. Activate the plugin and go to the Apearance -> Widgets screen.
3. The widged should be listed there and you can enable it into your blog.
4. Move the widget into the preferred order.
5. Enjoy the IPv4 Exhaustion Counter on your blog!

= The Half-Automated Installation =

1. Download the .zip package from Wordpress.org
2. Go to the plugin page in the admin pane. Upload the .zip file and install the plugin.
3. Activate the plugin and go to the Apearance -> Widgets screen.
4. The widged should be listed there and you can enable it into your blog.
5. Move the widget into the preferred order.
6. Enjoy the IPv4 Exhaustion Counter on your blog!

= The Manual Installation =

1. Download the .zip package from Wordpress.org
2. Upload it to your server and place the .zip file into the `/wp-content/plugins/` directory.
3. Unzip the .zip file
4. Go to the plugin page in the admin pane. Activate the plugin.
5. Go to the Apearance -> Widgets screen.
6. The widged should be listed there and you can enable it into your blog.
7. Move the widget into the preferred order.
8. Enjoy the IPv4 Exhaustion Counter on your blog!
